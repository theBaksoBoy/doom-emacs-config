# doom-emacs-config
My config for Doom Emacs

prerequisites:
1. install Doom Emacs (duh)
2. make emacspeak work in Doom Emacs, by (on debian-based distros) doing `sudo apt install espeak-ng emacspeak`
3. install sqlite3 to make org-roam work, by (on debian-based distros) doing `sudo apt install sqlite3`
